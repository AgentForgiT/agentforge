# Coding Standards

Metadata:

- Status: Draft
- Last updated: 2026-06-28

## Principles

- Prefer readable code over clever code.
- Use explicit interfaces and dependency injection at boundaries.
- Keep provider-specific behavior inside provider adapters.
- Avoid hidden configuration.
- Add tests for user-facing behavior and cross-module contracts.
