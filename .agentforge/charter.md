# AgentForge Project Charter

Metadata:

- Status: Draft
- Phase: Genesis
- Applies to: AgentForge ecosystem
- Last updated: 2026-06-28

## Purpose

AgentForge exists to become the reference open platform for Agentic AI Engineering.

## Scope

AgentForge includes:

- OpenAI-compatible AI gateway development
- provider abstraction and routing
- agent orchestration
- IDE and agentic coding tool compatibility
- MCP tooling
- LLM and agent benchmarking
- engineering standards
- research and documentation
- automation workflows
- AI context portability

## Non-Goals

AgentForge should not become:

- a closed vendor-specific platform
- a pile of disconnected repositories
- a thin wrapper around one provider
- a feature-first project with weak governance
- a benchmark project without reproducibility discipline

## Success Criteria

AgentForge succeeds when engineers can clone the project, understand its architecture, run its examples, validate its behavior, and extend it across AI tools without relying on private chat history or hidden assumptions.
