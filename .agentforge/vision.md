## Purpose

AgentForge aims to define practical, open engineering patterns for building, testing, documenting, and operating agentic AI systems.

## Vision

AgentForge should evolve like durable open infrastructure: coherent governance, modular architecture, strong docs, reproducible benchmarks, and careful release discipline.

The project should make AI-assisted engineering more portable across tools by defining a shared context layer that can be read by humans, AI assistants, and automation.

## Long-Term Outcomes

- A vendor-neutral gateway for AI providers and local models
- A modular ecosystem for agents, workflows, MCP tools, integrations, and benchmarks
- A public handbook for agentic AI engineering
- A portable AI Context Specification for AI-native projects
- A disciplined open-source governance model
